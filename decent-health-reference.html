Decent Health

Personal health tracking app — weight, leg measurements, habits, health diary, and doctors directory. Built to replace a Google Sheets workbook with a purpose-built mobile-first web app.

Deployment Workflow

  1. Download updated file(s) from Claude
  2. Go to github.com/danielgomez909/decent-health
  3. Click the file → pencil icon to edit, or drag-and-drop to upload
  4. Add a short commit message → click Commit changes
  5. Vercel auto-deploys in ~30 seconds
  6. For icon updates: remove app from home screen and re-add via Safari → Share → Add to Home Screen
